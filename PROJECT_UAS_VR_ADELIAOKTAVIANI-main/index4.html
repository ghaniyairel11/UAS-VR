Baca dan Pahami Panduan Penggunaan Tur Virtual cafe ini!

1. Tekan menu "Masuk" untuk memulai eksplorasi virtual kafe.

2. Pilih tombol "VR" di pojok kanan bawah untuk menikmati tur virtual dalam tampilan penuh.

3. Perhatikan baik-baik! Di tengah layar virtual terdapat kursor lingkaran hitam yang bisa digunakan sebagai alat navigasi dan pengganti kursor laptop untuk memilih gambar lainnya.

4. Gunakan tombol "Kembali" berwarna biru untuk kembali ke halaman utama.

Tentang Tur Virtual Kafe

Ini adalah Tur Virtual Kafe-Kafe yang ada di kota Bandar Lampung. Dengan tur virtual ini, Anda dapat menjelajahi berbagai kafe menarik di kota ini, memahami konsep dan suasana unik masing-masing kafe, atau hanya untuk menemukan tempat baru untuk dikunjungi.
